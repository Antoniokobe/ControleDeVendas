# 🛒 Sistema Desktop de Controle de Estoque e Vendas

Este projeto foi desenvolvido com base em um curso da Udemy, com o objetivo de criar um sistema desktop completo 
para **controle de estoque, vendas e gestão empresarial**, utilizando **Java** e **MySQL**. O sistema inclui funcionalidades como 
gerenciamento de funcionários, clientes, produtos, fornecedores, histórico de vendas, controle de permissões e muito mais.

## 🎯 Funcionalidades principais

- Cadastro e gerenciamento de:
  - Funcionários
  - Clientes
  - Produtos
  - Fornecedores
- Controle de estoque com baixa automática
- Histórico de vendas por cliente e produto
- Sistema de autenticação com níveis de acesso (Usuário / Administrador)
- Preenchimento automático de endereço via WebService (consulta por CEP)
- Interface gráfica intuitiva com Java Swing
- Arquitetura baseada em **MVC (Model-View-Controller)**

## ⚙️ Tecnologias utilizadas

- Java 17+
- MySQL
- JDBC
- Java Swing (GUI)
- WebService via API de CEP
- IDE: IntelliJ IDEA / Eclipse
- Git e GitHub
